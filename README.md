# Team Task Manager

## Description
This is a Team Task Manager web application built using HTML, CSS, and JavaScript. It allows users to create tasks, assign them to team members, and track their status with a simple dashboard.

## Features
- User login (Admin/Member role simulation)
- Create and assign tasks
- Task status tracking (Pending, In Progress, Completed)
- Dashboard showing total, completed, and pending tasks
- Delete tasks
- Data persistence using localStorage

## How to Use
1. Enter a username and login
   - "admin" → Admin role
   - other names → Member role
2. Enter task name and assign it to a user
3. Select task status
4. Click **Add Task**
5. View tasks in the list
6. Dashboard updates automatically
7. Click **X** to delete a task

##  Dashboard
- Total tasks
- Completed tasks
- Pending tasks
